ماذا ستتعلم؟

خلال هذه الدورة الشاملة، ستتعلم كل ما تحتاجه لتصبح مطور واجهات أمامية محترفًا، بدءًا من الأساسيات وصولًا إلى التقنيات المتقدمة:

1. HTML و CSS

  • بناء صفحات الويب باستخدام HTML.
  • تصميم وتنسيق المواقع باستخدام CSS.
  • إنشاء تصميمات متجاوبة تعمل على جميع الأجهزة.

2. JavaScript

  • تعلم أساسيات البرمجة باستخدام JavaScript.
  • التفاعل مع عناصر الصفحة باستخدام DOM.
  • إدارة أحداث المستخدم مثل النقر والإدخال.

3. React.js

  • بناء واجهات تفاعلية باستخدام مكتبة React.js.
  • إدارة البيانات باستخدام State Management.
  • التفاعل مع واجهات برمجة التطبيقات (APIs).

4. أدوات المطورين

  • استخدام Git و GitHub لإدارة المشاريع.
  • تحسين بيئة التطوير باستخدام Webpack و Babel.

بنهاية الدورة، ستتمكن من تصميم وتطوير مواقع ويب وتطبيقات تفاعلية احترافية!
